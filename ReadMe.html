Introduction:
Welcome to Domain Manager Pro. A program to track and manage your domain names and return information result information from various sources.

Features:

  • Domain database to store information on your domains
  • Page Rank checker, Alexa Rank, Google, MSN, Yahoo Link results
  • Whois checker.
  • System tray icon for easy access
  • Checks if domains are expiring within a time frame that you have set.
  • Export the information to csv files.
  • Easy to use interface.
  • Reports for Domains, Results, and Alexa Information.
  • And more!
